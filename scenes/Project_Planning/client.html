2. Solution Overview

  • 3CX IP PBX (Internet Protocol Private Branch Exchange), voice solution planned to be implememnted. The new phone system will route and manage calls within supply chain business units and dealership sectors through secure internet utilization.
  • The new 3CX voice solution will allow for, but is not limited to, the following features:
  • Call Forwarding
  • Call Queing
  • Autto Attendance
  • Reliable and Secure
  • Cost Saving
  • Integration with varius devices like IP phone and softphones
  • Accessable across different locations
  • Key aspects used for real-time voice communication includes:
  • VoIP - Voice over Internet Protocol
  • SIP - Session Initiation Protocol
  • Technology:
  • Voice data will be transmited as digital packets by leveraging the inernet protocal (IP).
  • This will allow for easier scalability and flexibility in the network setup.
  • Componants:
  • IP PBX server: The 3CX server will be the centrl unit that handles call management and routing.
  • SIP trunks: ICTGlobe will provide the virtual phone lines that will connect the IP PBX to the public telephone network.
  • 3CX Extensions: ICTGlobe will configure the user extensions that will be used to manage call groups and call flows effectively.
  • IP Phones: As per the set out solution proposal, SNOM handsets amd desk IP phones will be used for the majority of the planned solution, with the exception of a view Yealink devices.
  • Softphones: The 3CX softclients will be installed on user devices including computers and mobile phones, as and when required.
